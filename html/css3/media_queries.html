CSS3 Media Queries

媒体类型

Media Type媒体类型一般有all, screen, print等

页面中引用媒体类型的方式

1. link方式引入

<link rel="stylesheet" media="screen and (max-width: 600px)" href="small.css" />

2. xml方式引入

<?xml-stylesheet rel="stylesheet" media="screen" href="css/style.css" ?>

3. @import方式引入

@import url("css/reset.css") screen;
@import url("css/print.css") print;

3. @media方式引入

@media screen and (max-width:600px){
	...这里是样式语句...
    .page_left{display:none;}
}

媒体特性

属性 值 Min/Max 描述
color int yes 每种色彩的字节数
color-index int yes 色彩表中的色彩数
device-aspect-ratio int/int yes 宽高比例
device-height length yes 设备屏幕的输出高度
device-width length yes 设备屏幕输出的宽度
grid int no 是否是基于格栅的设备
height length yes 渲染界面的高度
width length yes 渲染界面的宽度
monochrome int yes 单色帧缓冲器中每像素字节
resolution 分辨率("dpi/dpcm") yes 分辨率
scan Progressive interlaced no tv媒体类型的扫描方式
orientation Protrait/landscape no 横屏或竖屏

用法

1.最大宽度Max Width

<link rel="stylesheet" media="screen and (max-width:600px)" href="small.css" type="text/css" />

2.最小宽度Min Width

<link rel="stylesheet" media="screen and (min-width:600px)" href="big.css" type="text/css" />

3.多个media queries使用

<link rel="stylesheet" media="screen and (min-width:600px) and (max-width:900px)" href="big.css" type="text/css" />

4.设置屏幕的输出宽度Device Width

<link rel="stylesheet" media="screen and (max-device-width: 480px)" href="iphone.css" type="text/css" />

5.针对iphone 4

<link rel="stylesheet" media="only screen and (-webkit-min-device-pixel-ratio: 2)" type="text/css" href="iphone4.css" />

6.针对iPad

<link rel="stylesheet" media="all and (orientation:portrait)" href="portrait.css" type="text/css" /> 
<link rel="stylesheet" media="all and (orientation:landscape)" href="landscape.css"  type="text/css" />

7.only关键字

<link rel="stylesheet" media="not print and (max-width: 1200px)" href="print.css" type="text/css" />

8.not关键字

<link rel="stylesheet" media="only screen and (max-width: 1200px)" href="print.css" type="text/css" />

9.逗号分格表示并列或者表示或,如下

<link rel="stylesheet" type="text/css" href="style.css" media="handheld and (max-width:480px), screen and (min-width:960px)" />
