CSS 3 摘要

盒类型box-sizing:

box-sizing:border-box;
-moz-box-sizing:border-box; /* Firefox */
 值可以是：content-box| border-box | initial | inherit

圆角边框

圆角边框(其实用border-radius就够了，基本上firefox,chrome,ie9都能识别)
-webkit-border-radius:10px;
-moz-border-radius:10px;
border-radius:10px;

单独圆角：
-webkit-border-top-left-radius:10px;
-moz-border-radius-topleft:10px;
border-top-left-radius:10px;

块元素阴影

（参数分别为rgba, 向右的偏移量，向下的偏移量，阴影大小）
-webkit-box-shadow: rgba(0,0,0,0.3) 0 1px 3px; /*Chrome, Safari*/
-moz-box-shadow: rgba(0,0,0,0.3) 0 1px 3px; /*Firefox*/
box-shadow: rgba(0,0,0,0.3) 0 1px 3px; /*IE9,Opera*/>

内阴影
box-shadow:5px 5px 5px #999 inset;

设置了阴影的扩展半径( spread )
box-shadow:5px 5px 5px 8px #999; 第四个参数就是扩展阴影半径

文字阴影

(参数为颜色，向右偏移量，向下偏移量，阴影大小)
text-shadow:black 1px 1px 3px;
text-shadow:rgba(50,50,50,.3) 1px 1px 3px;

过渡动画

过渡动画(参数分别为过度的属性color/background..., 过渡完成时间，过渡曲线，过渡的延迟执行时间)
transition:all .25s linear .1s;
多重变换用逗号隔开：
transition: color .25s linear, background-color .15s linear .1s;
变形曲线参数可为：ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier(x1, y1, x2, y2) 为自定义，x1,x2,y1,y2的值范围在[0, 1];

变形之旋转

transform:rotate(0deg);
-ms-transform:rotate(0deg); /* IE 9 */
-webkit-transform:rotate(0deg); /* Safari and Chrome */
-moz-transform: rotate(-45deg);
-o-transform: rotate(-45deg);

鼠标拖动时避免文字被选中

-webkit-touch-callout: none;
-webkit-user-select: none;
-khtml-user-select: none;
-moz-user-select: none;
-ms-user-select: none;
user-select: none;

Canvas设置宽高，一定要是canvas.width,和canvas.height.如果是设了canvas.style.width的话， 会像图片一样等比缩放。

让图片在firefox跟chrome中不可拖动

<img src="img/banner_01.jpg" draggable="false" />

输入框在chrome中有outline样式，设了外发光。可以在样式里重置为outline:none;来清除。

跨浏览器的背景色半透明

/*
* filter 渐变滤镜详细用法，参见这里
* StartColorStr 和 EndColorStr:
* #4c000000 是 30% 透明度的 #000000 的意思
* 组成: # + 透明度 + 颜色
* 算法: Math.floor(0.6 * 255).toString(16);
*/
 .rgba{
background:rgba(0, 0, 0, 0.3);
filter:progid:DXImageTransform.Microsoft.Gradient(GradientType=0, StartColorStr='#4c000000', EndColorStr='#4c000000');
}

content的值的设置

原来content不光可以是指定的字符串，也可以是变量。如在<a href="#" data-icon="3"> 链接上设了data-icon属性，那么就可能在content中引用这个值。content:attr(data-icon);
