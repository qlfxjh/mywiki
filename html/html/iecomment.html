IE条件注释

说明

(IE10版本不再支持)

条件注释的语法

Comment 类型 语法或可能的值
standard HTML comment <!-- Comment content -->
downlevel-hidden <!--[if expression]> HTML <![endif]-->
downlevel-revealed <![if expression]> HTML <![endif]>

条件表达式可以是组合的形式由以下元素构成：特殊字符，操作符，and/or value。下面表格显示了支持的字符。

Item Example Comment
IE [if IE] 用所有版本的IE浏览器查看都符合
value [if IE 7] IE后面跟一个数字，表示只符合这个版本号的IE。
WindowsEdition [if WindowsEdition] Windows Internet Explorer 8 on Windows 7。 字符串"WindowsEdition"是一个特殊字符用来表示用这个版本的window系统来查看网页时符合。
value [if WindowsEdition 1] WindowEdition后面跟一个数字，表示用这个版本的Window浏览网页时，符合条件
true [if true] 永远是真
false [if false] 永远是假

下面列出来了可以使用的操作符

Item Example Comment
! [if !IE] 否操作符，表示符合相反的情况。此表示式表示不是IE的所有浏览器都符合
lt [if lt IE 5.5] 小于操作符，当第一个参数比第二个参数小时，返回true。此实例表示低于5.5版本的IE浏览网页面符合
lte [if lte IE 6] 小于等于操作符. 此实例表示小于或等于IE6的版本浏览器符合
gt [if gt IE 5] 大于操作符。表示大于版本5的IE都符合
gte [if gte IE 7] 大于等于操作符
( ) [if !(IE 7)] 子表达式操作符，用于生成更复杂的表达式
& [if (gt IE 5)&(lt IE 7)] 与操作符
| [if (IE 6)|(IE 7)] 或操作符
